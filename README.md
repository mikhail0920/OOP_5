# Лабораторная работа №5. Итераторы и аллокаторы

## Постановка задачи
Целью работы является изучение устройства коллекций в стандартной библиотеке, а также получение навыков использования концепций «итератор» и «аллокатор».

Необходимо реализовать:
1.  Пользовательский класс управления памятью (`memory_resource`), реализующий конкретную стратегию работы с памятью.
2.  Контейнер (согласно варианту), использующий этот ресурс через `std::pmr::polymorphic_allocator`.
3.  Итератор для контейнера, соответствующий категории `std::forward_iterator_tag`.
4.  Модульные тесты с использованием **Google Test**.

### Описание варианта (№20)
Согласно таблице вариантов:
* **Контейнер:** Очередь (Queue).
* **Стратегия работы с памятью (№2):** Фиксированный блок памяти (выделяется один раз), информация о выделенных блоках памяти хранится в `std::vector`.
